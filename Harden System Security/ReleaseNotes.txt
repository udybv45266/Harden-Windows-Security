* Removed multiple dependencies by implementing them natively.

* Reduced the app package size by over 20MBs.

* Added a new backdrop option for the app called "Acrylic Thin", available in the app's settings page. This backdrop option provides a sleek and modern look to the app's interface, enhancing the overall user experience. It offers new options to further customize the Acrylic backdrop, such as controlling the tint color, tint opacity or luminosity opacity. You can view the changes you make to the Acrylic backdrop in real-time, allowing you to find the perfect settings for your preferences and your configurations will be saved for the next time you open the app.

* Improved keyboard navigation and accessibility: You can now use arrow keys on your keyboard to navigate between the buttons on the toolbars of each page.

* You can now press the ALT access key to reveal keyboard shortcuts on various user interface elements, helping with keyboard navigation and accessibility scenarios.

* You can now change the display mode of the Sidebar pane and switch between Overlay or Inline modes.

* The individual Apply/Verify/Remove buttons of each security measures have hover animations now.

* The Harden System Security app now responds to the Windows theme/color changes if the app's theme in the settings page is set to "Use System Setting".
